# DB-Schema-Sync

Скрипт на Python для синхронизации структуры базы данных между средами разработки (Test -> Prod). Формирует набор SQL-запросов для обновления целевой БД без удаления данных.

## Основной функционал
- Сравнение метаданных таблиц и колонок.
- Генерация команд `ALTER TABLE` для добавления и изменения полей.
- Автоматическое создание новых таблиц через `CREATE TABLE`.
- Безопасный режим: отсутствие команд `DROP` (колонки и таблицы на боевой базе не удаляются, даже если они удалены в тестовой).
